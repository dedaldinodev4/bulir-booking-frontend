## Overview
Web frontend of the Company Bulir booking application. Allows clients to book services, providers to manage their offerings, and admins to monitor the entire operation.

## Architecture
- Component-based Architecture (Next/React + TypeScript)
- Centralized state management with Redux/Zustand
- Routes protected based on roles (client, provider, admin)
- Communication with backend via REST API

## Technologies
- React
- Next
- Zod
- TypeScript
- TailwindCSS
- Axios
- React Query

## Running locally
1. Clone the repository
2. Copy `.env.example` to `.env`
3. Run `npm install`
4. Run `npm run dev`

## Tests
- `npm run test`
